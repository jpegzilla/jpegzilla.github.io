hello! I am a web designer, developer, and graphic designer from atlanta, georgia.

outside of the web, I don't really exist. the web is everything to me. I live here, and if I die nothing will be left of me except what I've built on the web.

nothing that exists in the world right now better represents me than the collection of digital artifacts I will create and gather here.

your website is a self-portrait written in code — it is you as much as you are you.

this particular website is an attempt to free my self from my mortal body — to create a place where I can make a lasting mark on the digital landscape. something that will outlast my human existence as I now know it. there's no erosion here.

thank you for visiting. if you do decide to contact me, I hope we can be friends. I'm sure we'll create something amazing.
